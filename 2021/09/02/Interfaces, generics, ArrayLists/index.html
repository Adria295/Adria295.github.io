Interfaces, generics, ArrayLists

Interface

  1. An interface is a set of one or more method signatures
  2. 一个class implement一个interface，就需要implement所有的method
  3. a formal way of telling Java that a class has certain functionality
  4. 一个class implement interface， 那么这个class的object可以存到这个interface的reference中

void:返回值类型修饰符，用来表明方法没有返回值

Important: 当把对象存在interface的reference里，只能用interface里的方法(toString除外)

原因是：“Each interface implicitly declares one method for each public method in Object”. //？？？

  • This includes toString, equals and getClass
    • You don’t have to re-implement them, but you can

ActionListener interface

  1. 在Swing里行为是一个ActionEvent
  2. ActionEvent被方法actionPerformed处理
  3. 如果想要例如点击鼠标发生事件，需要指定一个有actionPerformed方法的object给button
  4. ActionListener is the interface that defines actionPerformed

The ArrayList class

  1. 用来储存对象，但不是初始数据类型

  2. 自动填充

  3. It is a generic class. So

    ​ It has <> notation

    ​ when instantiated a type must be specified

  4. Constructors:

    ArrayList() - constructs an empty list with initial capacity 10

    ArrayList(int init) - constructs an empty list with intial capacity init

    • for(int i = 0; i<myList.size(); i++) System.out.println(myList.get(i));
