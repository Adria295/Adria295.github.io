Java Lecture2

Variables

  1. The data is stored somewhere in the computer’s RAM (random access memory)
  2. Java convention says variable names start with a lower case letter
  3. Variable names cannot start with numbers or “or”

Literals

  1. 字面常量(literal)是用于表达源代码中一个固定值的表示法
  2. 种类包括int / double / String / char

上述代码，int / String 是标识符(identifier)，因为它们都是用于标识该程序中的变量、类。

而666 / “Hello World!”是字面常量(Literals)，，因为它们都是在程序中直接以值的形式来操作和表现。

Mathematical

  1. Operations involving an int and a double will result in a double
  2. 1.0*a effectively converts a int into a double and then the division is done as a double division

Scope & passing variables to methods

  1. Variables declared in one method are not visible to other methods.

  2. When we call computeArea from main menu and pass some data to computeArea:

    computeArea creates its own variables and sets them to the values it is passed from main.

    the variables in computeArea can have the same name as those in main but they are different variables.

    This is very important!

    We say that main and computeArea are not in the same scope.

Compound Statements and Scope

  • A variable is visible in the block in which it is defined, and any blocks within that.
  • Outside the block it is invisible.

Methods: overloading

  • The Signature of a method consists of its name and argument list
  • Java allows us to have methods with the same name but different arguments.

Stopping a program

  • System.exit(int n)

  • Calling this method, terminates your program.

  • The value n is passed back to whatever started your program and the normal convention is:

  • N = 0; // successful termination – the program did what it wanted to do

  • N = -1; N = 1; // or any other number that isn’t 0: unsuccessful terminatione
