Programming Construcs

Abstract Class

  • An abstract class cannot be instantiated; only extended.
  • they sit at the roots of class hierarchies (have to be public)
  • abstract methods : An abstract class is used to group together classes that should have a common function(s) of different implementations.

抽象类总结规定

    1. 抽象类不能被实例化(初学者很容易犯的错)，如果被实例化，就会报错，编译无法通过。只有抽象类的非抽象子类可以创建对象。
    1. 抽象类中不一定包含抽象方法，但是有抽象方法的类必定是抽象类。
    1. 抽象类中的抽象方法只是声明，不包含方法体，就是不给出方法的具体实现也就是方法的具体功能。
    1. 构造方法，类方法（用 static 修饰的方法）不能声明为抽象方法。
    1. 抽象类的子类必须给出抽象类中的抽象方法的具体实现，除非该子类也是抽象类。

Visibility Modifiers

private: accessible only to objects of the current class. 􏰀 e.g. methods that will not be reused elsewhere

(default) – when no keyword is present: accessible by classes within the package.

protected: Accessible in inherited classes.

public: Accessible anywhere.

The static Keyword

方法和参数，方法的话不需要实例化就能调用，最好的例子Math类

IO

  • Two modes of files read/writes:

    • 􏰀 Text – Read/write character strings, e.g. the integer ‘16’ is stored as

      the string of characters ‘1’ and ‘6’.

    • 􏰀 Binary – Read/write byte values directly, e.g., the integer ‘16’ is stored

      as ‘0x00 0x00 0x00 0xA0’ (shown in hex notation).

Thread Priorities

Priority could be read using getPriority()

  • 􏰀 MIN_PRIORITY = 1

  • 􏰀 NORM_PRIORITY = 5 (default)

  • 􏰀 MAX_PRIORITY = 10

    Priority could be changed using setPriority(int)

􏰀 value between 1 to 10

Thread Creation

  • extend the Thread class
  • implement the Runnable interface
  • Note: we call Thread.start(), never call run()

Thread Control

  • yield() temporarily cause a thread to pause and allow other threads to execute.
  • Sleep() puts a thread to sleep for a specfied time

Thread Interruption

线程A可被线程B中断的两种情况：

  1. B is running an interruptable method (e.g. sleep)
  2. B’s interrupted status is set to TRUE

Thread Control, continued

join() allows you to stop the current Thread till another one ends

即join就是指等待该线程结束，然后才继续往下执行自身线程

Design Principles

The Composite Pattern（树形结构）

perform the same operation on objects or groups of objects

The 3 elements of composite:

1.Component: The highest level of abstraction, typically an interface.

2.Leaf: A class for individual objects

3.Composite:A class for groups of objects.

The Decorator Pattern

意图：动态地给一个对象添加一些额外的职责。就增加功能来说，装饰器模式相比生成子类更为灵活。

The Observer Pattern

Helps keep your objects up-to-date about information they care about.

  • The Subject – the class that contains (and publishes) the state

  • 􏰀 The Observer – an abstract class for objects that can subscribe to

    state updates

  • 􏰀 Concrete observers – any class that extends the Observer in order to

    be able to subscribe and unsubscribe to state updates
